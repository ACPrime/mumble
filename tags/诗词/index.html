7月19日天有点阴但是很舒服

197月

有一个周没有发日记了，时间过得真快。
再这样下去日记要变成“周记”啦。
这一个周都是在实验室度过的，回想起来也没做好什么东西，整天东看看西看看，真的是很愧疚。
有些不喜欢这样的生活、
生活总需要有目标，有激情，是吧。

昨晚在工科E座旁边昏黄的路灯下坐着天马行空，其实也好久没有这样静静地发呆了。

真的很喜欢好妹妹乐队的歌。

明天又是周一，就像学渣在新学期开始之前喜欢说的一样，“一定要好好努力了！”。May everything go well~

立冬

711月

古代诗人有不少描述立冬的诗作，明代人王稚登有一首《立冬》诗：“秋风吹尽旧庭柯，黄叶丹枫客里过。一点禅灯半轮月，今宵寒较昨宵多。”秋去冬来，树叶落尽，客居他乡的旅人唯有禅月相伴，孤寂中尤显寒意。
大诗人李白也有一首《立冬》，他写道：“冻笔新诗懒写，寒炉美酒时温。醉看墨花月白，恍疑雪满前村。”立冬之夜，笔墨都冻凉了，诗人只好与炉火琼浆相伴，微醉中竟将一地月光当成了雪迹……

If Winter comes, can Spring be far behind?
